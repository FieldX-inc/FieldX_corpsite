# 記事 QA チェックリスト

## 公開判定

- [ ] 公開可
- [ ] 要修正
- [ ] 公開不可

## SEO

- [ ] 主軸キーワードが不自然なく扱われている
- [ ] タイトルと導入が検索意図に合っている
- [ ] 見出し構造が論理的
- [ ] 内部リンク先が適切

## AIO / GEO

- [ ] 冒頭で短く結論を返している
- [ ] 定義や比較軸が明確
- [ ] 引用されやすい要約文がある
- [ ] 根拠の弱い主張を断定していない

## 正確性

- [ ] 数値、制度、比較表現に根拠がある
- [ ] 最新確認が必要な箇所が明示されている
- [ ] 競合や第三者への言及が安全

## 訴求

- [ ] 記事価値を壊さない CTA になっている
- [ ] 読者温度に合う導線になっている
- [ ] 売り込み口調が強すぎない

## 表現

- [ ] 日本語が自然
- [ ] 冗長な重複がない
- [ ] 見出しと本文の約束が一致している
- [ ] 公開前に埋める metadata が揃っている
