# 記事 QA チェックリスト

## 公開判定

- [ ] 公開可
- [x] 要確認のうえ公開候補
- [ ] 公開不可

## SEO

- [x] 主軸キーワードが不自然なく扱われている
- [x] タイトルと導入が検索意図に合っている
- [x] 見出し構造が論理的
- [x] 内部リンク先が概ね適切

## AIO / GEO

- [x] 冒頭で短く結論を返している
- [x] 定義や比較軸が明確
- [x] 引用されやすい要約文がある
- [x] 根拠の弱い主張を断定していない

## 正確性

- [x] 法務、制度、個人情報に最新確認が必要な箇所を明示している
- [x] 最新確認が必要な箇所が明示されている
- [x] 競合や第三者への言及が安全

## 訴求

- [x] 記事価値を壊さない CTA になっている
- [x] 読者温度に合う導線になっている
- [x] 売り込み口調が強すぎない

## 表現

- [x] 日本語が自然
- [x] 冗長な重複がない
- [x] 見出しと本文の約束が一致している
- [ ] 公開前に埋める metadata が揃っている

## 判定

- 判定: 要確認のうえ公開候補

## 重大

- なし

## 中

- 督促の自動完結に読めないか、公開前に確認したほうがよい
- 法務、制度、個人情報の表現は公開直前に再確認したほうがよい
- `入金消込` 記事との役割分担を、公開版でもう一度確認したほうがよい

## 軽微

- タイトル案 1 はやや長いため、公開時に 32 文字前後へ調整してもよい
- meta description は microCMS 投入時に 120 字前後へ圧縮してもよい

## 公開前確認事項

- `microCMS` の公開済み記事一覧 live 確認を再実施する
- 法務、制度、個人情報の最新内容を確認する
- 本文中の `最新確認が必要` 箇所を公開前に社内レビューする

## 残課題

- title 最終版
- description 最終版
- category
- eyecatch
- publishedAt
- slug 最終確認

## 修正メモ

- 重大: なし
- 中: 督促の自動完結に見えないよう表現を最終確認
- 軽微: title と description の文字数調整
