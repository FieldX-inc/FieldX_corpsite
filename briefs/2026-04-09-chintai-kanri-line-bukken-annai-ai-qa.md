- [x] 冒頭で短く結論を返している

## 正確性

- [x] 数値、制度、比較表現に根拠がある

- [x] 記事価値を壊さない CTA になっている

## 表現

- [x] 日本語が自然

- [ ] 公開前に埋める metadata が揃っている

- 重大:

- 中:

  - microCMS description の表示長に合わせて 120 字前後へ微調整する
  - アイキャッチ案を追加する
